####智能社视频材料 - 初探JavaScript魅力 - 1
* 编写JS的流程
	* ⒈布局：HTML+CSS
	* ⒉属性：确定要修改哪些属性
	* ⒊事件：确定用户做哪些操作（产品设计）
	* ⒋编写JS：在事件中，用JS来修改页面元素的样式
* 定义和调用
	* 函数定义：只是告诉系统有这个函数，不会实际执行
	* 函数调用：真正执行函数里的代码
* 第一个JS兼容性问题
	* document.getElementById在任何浏览器下均可使用

* while循环
	* while循环结构：
	``` javascript

		while(条件){
			语句
		}
	```
	* while循环实例：只要i小于5，将循环不断地执行循环体内的语句
	``` javascript
		var i = 0;
		while(i<5){
			alert(i);
			i++;
		}
	```
